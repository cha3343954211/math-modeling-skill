# 无人机低空航线优化与空域调度建模案例

## 适用题型
- 无人机/无人车路径规划、航线优化
- 低空空域管理、空域网格化划分
- 多机协同调度、冲突规避
- 低空经济、智慧城市巡检类题目

## 题目特征
- 给定多类作业场景（农田植保、河道巡检、城区治理、电力巡检等）
- 多个固定起降点（机场）坐标
- 无人机参数：续航、速度、覆盖宽度、能耗
- 安全约束：水平/垂直间距、禁飞区、每网格最大架数
- 优先级设定（防汛 > 电力 > 城区 > 农业等）

## 建模方法映射

### 问题一：空域网格化划分与评价
- **方法**：AHP层次分析法 + 网格剖分 + 综合评分分级
- **指标体系**：区域重要度、作业密集度、空域冲突风险度
- **关键**：AHP一致性检验 CR<0.1
- **输出**：网格等级(A/B/C)、最大承载无人机数、各片区资源利用率
- **利用率计算**：用实际分配无人机数 vs 单次全覆盖需求架数，而非网格承载力
  - 单架覆盖面积 = 速度 × 续航 × 覆盖宽度
  - 需求架数 = ceil(区域面积 / 单架覆盖面积)
  - 利用率 = 分配架数 / 需求架数（上限1.0）

### 问题二：单无人机航线优化
- **农田（矩形区域）**：平行扫描航线（蛇形/往返），目标=全覆盖+低能耗+低重复率
- **河道（狭长带状）**：沿长轴方向平行扫描，需多条带覆盖宽度方向
- **Baseline 必备**：往返直线航线 vs 蛇形连续航线 vs 最优方向航线
- **覆盖能力计算**：
  - 最大飞行距离 = 速度 × 续航时间
  - 有效作业距离 = 最大距离 - 起降转场距离(来回)
  - 最大覆盖面积 = 有效距离 × 覆盖宽度
  - 辐射边界 = 最大距离 / 2（单程）

### 问题三：多无人机协同调度
- **垂直分层**：按优先级分配飞行高度层（高层=高优先级）
- **时序调度**：分时段（如每30min一槽），高优先级全时段、低优先级错峰
- **冲突检测**：区域重叠分析，重叠区按优先级仲裁
- **动态避让**：空域饱和、多任务叠加、临时增派、紧急避障四类场景
- **多目标**：覆盖率最大 + 时长最短 + 冲突最低 + 能耗均衡
- **约束**：水平间距、垂直间距、每网格最大架数、续航、优先级

### 问题四：综合效能评估
- **四维度**：作业效率、飞行安全、能耗成本、覆盖质量
- **方法**：多层指标体系 + 极差归一化 + 加权综合评分
- **对比**：传统人工 / 随机航线 / 优化调度 三模式
- **输出**：雷达图 + 柱状图 + 短板分析 + 优化方案

## 常见坑
1. 利用率公式：不要用"网格承载力"做分母，要用"单次全覆盖需求架数"
2. matplotlib barh alpha 不能是列表，要用循环逐bar绘制
3. LaTeX表格列数：数 `&` 个数 + 1 = tabular列定义数
4. 蛇形航线总距离 = 主飞行距离 + 连接段距离，不要漏掉连接段
5. 覆盖面积对续航高度敏感（非线性关系），灵敏度分析时注意
6. 冲突区域要画出来（重叠矩形），不能只文字描述

## 代码组织
```
quest1/codes/font_setup.py  ← 共享中文字体模块（复制到每个quest）
quest1/codes/quest1_xxx.py
quest2/codes/quest2_xxx.py
quest3/codes/quest3_xxx.py
quest4/codes/quest4_xxx.py
```
font_setup.py 用 `font_manager.addfont()` 显式加载 msyh.ttc/simhei.ttf。
每个 quest 脚本独立运行，`sys.path.insert(0, os.path.dirname(__file__))` 导入同目录 font_setup。
