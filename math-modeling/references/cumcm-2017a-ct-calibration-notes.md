# CUMCM 2017A CT系统参数标定及成像案例记录

## 适用场景

题型：CT/Radon 投影、系统参数标定、滤波反投影成像、模板设计与稳定性分析。

触发关键词：`2017A`、`CT系统参数标定`、`Radon`、`滤波反投影`、`problem2.xls/problem3.xls`、`512×180 接收信息`。

## 输入资产

典型文件：
- `CUMCM-2017-problem-A.docx`：题面。
- `A题附件.xls`，含 5 个 sheet：
  - 附件1：`256×256` 标定模板吸收率矩阵；
  - 附件2：`512×180` 模板接收信息；
  - 附件3：`512×180` 未知介质1接收信息；
  - 附件4：`10×2` 指定点物理坐标；
  - 附件5：`512×180` 未知介质2接收信息。

## 推荐建模路线

1. **输入预检**
   - 在用户指定题目目录下原地创建 `支撑材料/` 标准结构。
   - 将题面、附件复制/引用到 `支撑材料/data/`。
   - 审计各 sheet 维度、缺失、数值范围。

2. **Baseline**
   - 默认旋转中心为图像中心，角度为 `0°..179°` 等间隔，直接 FBP 重建。
   - 作为对照，不应直接作为最终参数标定结果。

3. **主模型：模板 Radon 投影匹配标定**
   - 对附件1模板做 `skimage.transform.radon`，得到理论投影库。
   - 对附件2每列实测投影，搜索候选角、探测器尺度、中心偏移，使标准化相关系数最大：
     `corr((y-mean)/std, (p-mean)/std)`。
   - 用 CT 连续旋转约束拟合角度序列，不要机械采用每列独立最优角；模板对称性会造成局部跳角。
   - 探测器间距可由理论投影到探测器单元的尺度中位数估计：`d_pix = 1 / scale`。
   - 中心偏移可用近似正弦关系拟合：`b_j ≈ scale*(dx*cos(theta_j)+dy*sin(theta_j)) + c`。

4. **未知介质重建**
   - 用标定角度对附件3、附件5分别做 `iradon(..., filter_name='ramp', output_size=256)`。
   - 对重建结果做非负截断；必要时对极端振铃做高分位裁剪，但论文中要说明这是数值伪影处理。
   - 输出完整 `problem2.xls`、`problem3.xls`，并建议同步输出 `.xlsx` 方便检查。

5. **指定点吸收率**
   - 若托盘按 `100 mm × 100 mm` 处理，像素尺寸为 `100/256 mm`。
   - 附件4物理坐标 `(x,y)` 到矩阵坐标：
     - `col = x / pixel_mm - 0.5`
     - `row = 255.5 - y / pixel_mm`
   - 用双线性插值读取点值，避免最近邻在像素边界处跳变。

6. **几何信息**
   - 用 Otsu 或分位阈值分割重建图。
   - 连通域输出面积、重心、边界框、平均/最大吸收率。
   - 对小孤立组件要谨慎：FBP 边缘振铃可能产生伪组件，论文中应以主体连通域解释几何形状。

7. **检验与稳定性**
   - 模板回代：用标定参数重建附件2，并与附件1比较 MAE/RMSE/SSIM。
   - 参数扰动：至少扰动初始角和探测器间距，记录 RMSE/SSIM 变化。
   - 新模板设计：建议“非中心大圆盘 + 环形薄边 + 多方向矩形条 + 灰度阶梯块”，目标是减少角度对称歧义并分离几何误差和增益误差。

```python
from skimage.transform import radon, iradon
from scipy import ndimage

rad = radon(template_img, theta=np.arange(180), circle=False)
rec = iradon(sinogram, theta=angles, output_size=256,
             filter_name='ramp', circle=False)
rec = np.clip(rec * amplitude_scale, 0, None)

# 指定点双线性插值
pixel_mm = 100 / 256
col = x / pixel_mm - 0.5
row = 255.5 - y / pixel_mm
value = ndimage.map_coordinates(rec, [[row], [col]], order=1, mode='nearest')[0]
```

## 论文叙事重点

- 本题不要只写“使用 Radon/FBP”；必须写清楚：
  1. 接收信息为什么是线积分；
  2. 已知模板如何用于标定角度、间距和中心；
  3. 标定参数如何传递到附件3/5；
  4. `problem2.xls`、`problem3.xls` 如何由模型生成；
  5. 10点吸收率如何由物理坐标插值得到；
  6. 稳定性与新模板如何对应第四问。

## 常见坑

- **中文路径 workdir**：终端工具可能拒绝含中文字符的 `workdir`。可在简单工作目录运行 Python，并在脚本中使用绝对路径指向题目目录。
- **角度局部跳变**：每列独立相关最优角可能被模板对称性误导，需加入连续旋转/等步长约束。
- **FBP 伪影误判**：边缘振铃会产生小连通域，不应把所有小斑点都解释成真实结构。
- **`.xls` 写出**：若缺少 `xlwt`，可输出制表符分隔的 `.xls` 兼容文本，同时另存 `.xlsx`；支撑材料中说明格式。
- **论文篇幅**：CT题如果正文偏短，优先补充数据审计、参数匹配表、矩阵摘录、算法伪代码、误差来源和提交文件清单。
