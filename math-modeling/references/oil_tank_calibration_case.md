# 储油罐变位识别与罐容表标定案例复盘

适用：CUMCM 2010A 类“卧式储罐/容器变位识别/罐容表标定”问题，以及任何“几何模型 + 检测数据标定”的数学建模题。

## 初稿容易低质的原因

- 只写几何积分模型，忽略题图尺寸、探针位置人工读取带来的系统误差。
- 只输出一条罐容表，没有说明“理论几何表”和“实测数据校正表”的差异与用途。
- PCHIP/插值罐容表若不加入空罐、满罐物理边界，低油位或高油位可能出现不合理平台。
- 灵敏度分析只扰动一个参数，显得薄弱；至少应分析纵向倾角和横向偏角两个关键参数。
- 附件有进/出油量时，未用相邻时刻显示容积差与净进出油量做数据一致性检验。

## 推荐改进流程

1. **实验罐数据反演**
   - 将进油记录换算为 `V_i = V_0 + Q_i`。
   - 将出油记录换算为 `V_i = V_s - Q_i`，其中 `V_s` 可由同姿态进油曲线估计第一条出油记录对应容量后加回第一条累加出油量得到。
   - 加入物理边界：空罐 `(0,0)`，满罐 `(H_max,V_full)`；避免外推平台。
   - 使用 PCHIP 单调插值生成 1cm/10cm 罐容表，不要用会振荡的高阶多项式。

2. **实际罐几何模型**
   - 对圆柱 + 球冠组合罐，沿罐长离散积分圆缺面积：
     `V(h, α, β)=1000∫ A(y(x); r(x)) dx`。
   - 圆缺面积：
     `A(y;r)=r^2 arccos((r-y)/r) - (r-y) sqrt(2ry-y^2)`。
   - 纵向倾角可写入局部液深：`y(x)=h cosβ + tanα (x_p-x)`（若探针偏心信息不足，这是可解释的一阶近似）。

3. **参数识别 + 数据校正双表**
   - 用非线性最小二乘估计 `α, β`，报告 RMSE、MAE、MAPE。
   - 若几何模型残差较大，不要硬说模型很好；说明误差来自题图尺寸、探针坐标和横向偏转简化。
   - 同时用实测油高--容积点建立 PCHIP 数据校正曲线；最终表可同时列“几何模型标定值”和“数据校正标定值”。
   - 表中对低于/高于实测油高范围的延拓值应标注“边界延拓，可信度较低”。

4. **检验与灵敏度**
   - 若附件有进/出油量，做流量差分检验：`ΔV_i ≈ q_i^{in}-q_i^{out}`，报告 RMSE/MAE。
   - 至少扰动两个关键参数（如 `α`、`β`），范围可取估计值附近 ±1° 或 ±10%~±20%。
   - 给出灵敏度图和表，比较哪个参数更敏感，并给现场复测建议。

## 论文写作要点

- 摘要必须包含：变位影响的具体 L 数值、识别出的 `α/β`、拟合误差、检验误差、灵敏度结论。
- 问题分析中明确说明为什么采用“实验反演 + 几何积分 + 数据校正”而不是单一模型。
- 模型评价要承认几何近似的局限，并把数据校正作为工程实用补救，而不是掩盖残差。
- 最终支撑材料应包含：求解脚本、1cm/10cm 标定表、残差数据、灵敏度数据、冻结数字 JSON、方法说明。
