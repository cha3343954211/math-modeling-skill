# CUMCM 2010A 储油罐变位识别与罐容表标定经验

用于以后处理“卧式储油罐 / 罐容表 / 变位识别 / 油位-容量标定”类题目时参考。

## 数据读取与题面提取

- 旧国赛题常见 `.doc` + `.xls`：题面可用 `antiword` 快速转文本；Excel 97-2003 `.xls` 需 `xlrd` 支持，必要时先安装/确认依赖。
- 读取附件后必须打印所有 sheet 的表头、前几行、末几行，确认“累加进油量/累加出油量”的含义和初始油量说明。
- 不要直接把出油累加量当作绝对体积；出油数据需先推断出油起点体积，再做 `V_i = V_start - Q_out_i`。

## 小实验罐（问题一）稳健流程

1. 将进油表用题注初始油量换算为绝对体积：`V = V0 + cumulative_in`。
2. 出油表先用同工况进油曲线估计第一条出油记录油高处体积，再加回第一条累加出油量，得到出油起点体积。
3. 合并进/出油样本，按油高排序，同油高取均值。
4. 用 PCHIP/保形单调插值生成罐容表；不要用全局高次多项式。
5. 对实验数据覆盖范围外的油高，若没有可靠几何尺寸，不要让 PCHIP 外推；应裁剪到最近实验端点或明确标注“外推/端点保持”。本次出现过外推爆炸，根因是对倾斜实验低油位段无数据却直接 extrapolate。
6. 论文中比较“倾斜-无变位”的差异时，只在两组数据共同覆盖的油高区间统计最大差和平均差。

## 实际罐（问题二）建模注意

- 建立几何积分模型：沿罐长方向离散，计算每个截面的圆缺面积并积分；球冠端可通过轴向半径函数处理。
- 若题图尺寸/探针位置识别不充分，应在论文中明确“按图读取/近似”的尺寸假设，并把由此带来的误差写入模型不足。
- 附件2同时有显示油高、显示容积和进/出油量：
  - 显示容积可作为拟合观测值；
  - 进/出油量更适合做相邻差分一致性检验；
  - 用累计流量重构绝对体积时要对齐第一条记录，通常先令第一条观测作为基准，再使用后续净流量累积。否则会把第一条事件重复计入，导致尾部真实体积变成负数。
- 参数识别应输出：变位参数、RMSE/MAE/MAPE、残差图、标定表、灵敏度分析。
