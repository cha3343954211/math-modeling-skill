# CUMCM 2014D 储药柜设计案例记录

## 适用题型
- 储药柜/货架/药槽设计；
- 带“间距类型数量少 + 空间冗余小”双目标的离散优化；
- 药盒、包装盒等长方体物品在槽/格中的几何适配与容量估算。

## 本题关键转化
2014D 的核心不是连续拟合，而是“可行间距区间覆盖 + 类型数/冗余折中 + 货架装箱”。

### 可行槽宽/槽高区间
对药盒宽度 `w_i`：
- 必要间隙：槽宽下界 `a_i = w_i + 4`（左右各 2 mm）；
- 防并排重叠的保守上界可取 `b_i = 2*w_i + 4 - eps`；
- 不要机械把 `槽宽 < 药盒长度 l_i` 作为硬约束：个别药盒会因此变成不可行。若要处理水平旋转，优先在论文中说明由推送槽结构和防并排上界共同控制，或将长度约束作为可选敏感性假设而非默认硬约束。

对药盒高度 `h_i`：
- 槽高下界 `c_i = h_i + 4`；
- 防叠放可用 `2*h_i + 4 - eps` 作保守上界。

## 问题一：最少竖向隔板间距类型
建模为一维区间刺点问题：每种药品有可行槽宽区间 `[a_i,b_i]`，一种槽宽类型是落在若干区间交集内的点。

算法：
1. 按右端点 `b_i` 升序排序；
2. 对第一个未覆盖区间，选择其右端点作为槽宽类型；
3. 覆盖所有包含该点的区间；
4. 重复直到全部覆盖。

这是区间刺点最少点数的贪心最优算法。输出每个类型对应的药品编号。

## 问题二：宽度冗余与类型数折中
给定类型数 `K`，按槽宽下界排序后做连续分组动态规划：

- 分组 `[u,v]` 的槽宽取 `S_uv=max(a_i)`；
- 若 `S_uv <= min(b_i)`，分组可行；
- 分组成本 `C(u,v)=sum(S_uv-a_i)`；
- 状态 `F(k,j)=min_i F(k-1,i)+C(i+1,j)`。

对每个 `K` 得到最小总宽度冗余曲线，再用归一化综合评分或膝点选择合理类型数。必须同时报告：
- baseline：每个实际宽度一种类型、或 `K=1` 极少类型方案；
- 折中曲线图；
- 每类槽宽、药品数量、宽度范围、总/均值冗余；
- 完整药品编号分组文件。

## 问题三：横向隔板间距与平面冗余
在问题二宽度分组基础上做高度动态规划。

目标不是单纯高度冗余，而是题面定义的平面冗余：

`P = sum((slot_width_i - (w_i+4)) * (slot_height_i - (h_i+4)))`

实现时高度分组成本可用宽度冗余作权重。为避免精确零宽度冗余样本让高度优化退化，可在优化权重中加极小正数（如 `0.05`），但最终报告必须用真实平面冗余重新计算。

## 问题四：储药槽个数与柜数
每种药品单槽容量：

`q_i = floor(1500 / l_i)`

每天集中补药一次时所需槽数：

`n_i = ceil(D_i / q_i)`

柜体数量估算可采用货架行结构：
1. 同一高度类型形成若干行；
2. 每行宽度上限 2500 mm；
3. 某高度类型所需行数 `ceil(sum(n_i * slot_width_i) / 2500)`；
4. 将所有行按高度用 First-Fit Decreasing 装入有效高度 1500 mm 的柜体。

同时给出面积下界与行高下界作为验证：
- 面积下界：`ceil(sum(n_i*S_i*T_i)/(2500*1500))`；
- 行高下界：`ceil(sum(rows_t*T_t)/1500)`。
