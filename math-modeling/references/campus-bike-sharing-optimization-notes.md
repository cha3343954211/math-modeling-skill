# 校园共享单车优化案例（2026校赛A题）

## 题目概况

某城市大学东西校区6个核心停车点位的共享单车优化问题，含4个子问题：
1. 连续时间ODE模型（平衡点/稳定性/分段求解）
2. 离散差分方程24小时预测
3. 跨校区双目标优化调度
4. 特殊场景（雨天/考试周）与参数灵敏度分析

## 模型体系

### ODE模型：dN/dt = aN - bN² - c

**Bernoulli型非线性ODE**，适用于任何"存量驱动+饱和衰减+基础消耗"的动力学系统。

**平衡点**：N* = (a ± √(a²-4bc)) / (2b)
- 判别式 Δ = a²-4bc > 0 时存在两个实数平衡点
- 较小平衡点 N₁*：不稳定（dF/dN = √Δ > 0）
- 较大平衡点 N₂*：稳定（dF/dN = -√Δ < 0）

**数值求解**：四阶Runge-Kutta，分时段（高峰/平峰）分段拼接。关键：函数签名用 `t_start, t_end` 独立参数，不用 tuple。

### 差分方程模型：N_{k+1} = αN_k - βN_k² + γ

离散化对应关系：α ≈ 1+aΔt, β ≈ bΔt, γ ≈ cΔt。以1小时为步长迭代，按时段切换参数。

### 双目标优化调度

- 目标1：min 拥堵时长 Σ 1[N_{k,i} > 200]
- 目标2：min 调度车辆总数
- 约束：仅平峰调度、车辆守恒(ΣN=800)、非负、合理区间[30,200]
- 求解：有限空间约束优化搜索（调度时刻×调出点×调入点×数量）

### 灵敏度分析

- α参数（增长系数）：灵敏度高，低存量点位最敏感（相对变化大）
- γ参数（基础增量）：灵敏度极低，模型高度稳健

## 关键结果

| 指标 | 数值 |
|------|------|
| ODE西校宿舍区稳定点(平峰) | 290辆（远超200阈值） |
| 差分方程拥堵点位 | 西校宿舍区全天25h均超200辆 |
| 调度方案 | 凌晨3:00调35辆 + 中午13:00调10辆 |
| 调度改善率 | 拥堵时长 25→13 点·时（-48%） |
| 雨天场景 | 供给不足122点·时，需减少调出 |

## 踩坑记录

1. **RK4函数签名**：`solve_ode_rk4(N0, t_span, a, b, c)` 中 t_span 是 tuple，内部 `np.arange(t_span[0], t_span[1])` 报 `TypeError: 'float' object is not subscriptable`。改用 `t_start, t_end` 两个参数即可。
2. **总量守恒验证**：调度后必须逐时验证 ΣN=800，否则调度逻辑有bug。
3. **雨天场景陷阱**：α×0.7后全校借车意愿下降，之前设计的"从西校宿舍区调出"方案会导致供给不足。雨天需要保守调度甚至反向调入。
