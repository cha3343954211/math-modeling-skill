# CUMCM 2011C 养老金制度改革案例记录

## 适用场景

当题目涉及企业职工基本养老保险、养老金替代率、基金缺口、退休年龄政策、工资预测与现金流平衡时，可参考本记录。该记录来自一次完整完成 2011 年 C 题“企业退休职工养老金制度的改革”的建模交付。

## 题目拆解

- 问题一：利用历史社会平均工资预测未来工资。推荐先设 baseline（如近十年 CAGR 指数外推），再采用更符合长期经济规律的增长率收敛模型，避免远期指数爆炸。
- 问题二：用企业分年龄段工资区间频数估计各年龄段平均工资，并除以企业整体平均工资得到缴费指数；再根据基础养老金和个人账户养老金公式计算替代率。
- 问题三：建立缴费期基金累计值与退休后养老金领取现金流模型，计算领取至指定死亡年龄时的缺口，以及累计领取额首次达到累计缴存基金的平衡年龄。
- 问题四：不要只定性谈改革；至少给出一个“单一政策变量”的定量不可行性或敏感性证据，再提出组合改革措施。

## 可复用模型结构

### 工资预测

主模型可设为增长率指数衰减收敛：

```text
g_t = g_inf + (g_0 - g_inf) exp(-k(t - t0))
W_t = W_{t-1}(1 + g_t)
```

其中 `g_0` 可取近 5 年或近 6 年平均增长率，`g_inf` 表示长期名义工资增长率，`k` 控制收敛速度。论文中必须解释该模型比固定 CAGR 更适合远期预测。

### 缴费指数

对工资区间数据，用区间组中值估计平均工资：

```text
xbar_j = sum_h n_jh * midpoint_h / sum_h n_jh
I_j = xbar_j / xbar_enterprise
```

如果题目要求某职工从 2000 年起缴费，且附件工资分布为 2009 年，则用“2009 年年龄 = 起缴年龄 + 9”匹配年龄段。

### 养老金公式

```text
基础养老金 = (上年度省月均工资 + 本人指数化月均缴费工资) / 2 * 缴费年限 * 1%
本人指数化月均缴费工资 = 上年度省月均工资 * 平均缴费指数
个人账户养老金 = 个人账户储存额 / 计发月数
替代率 = 退休初始月养老金 / 退休前本人月工资
```

个人账户储存额可按年递推：

```text
C_{t+1} = C_t(1+r) + 0.08 * I * W_t
```

### 基金缺口

若从单个职工对应现金流角度计算，可将企业缴费 20% 与个人缴费 8% 合计作为基金缴存：

```text
F_R = sum_{t=start}^{retire-1} 0.28 * I * W_t * (1+r)^{retire-1-t}
Y_y = 12 * pension_month0 * (W_y / W_{retire-1})^alpha
Gap_75 = sum_{y=retire}^{death-1} Y_y - F_R
```

累计领取额首次超过 `F_R` 的年龄可用年内线性插值作为收支平衡年龄。

## 论文写作要点

- 摘要必须给具体数字：工资预测关键年份、不同退休年龄替代率、基金缺口、平衡年龄、政策敏感性结果。
- 问题四要用表格证明“单一提高缴费率不可行”或“延迟退休/调整系数有效”，再提出组合改革。
- 明确说明局限：分组工资用组中值、缴费指数固定、单个职工现金流不等于宏观统筹精算、死亡年龄固定。

## LaTeX/PDF 坑

自动生成表格时，表头如 `替代率/%`、`所需总缴费率/%` 必须写成 `替代率/\\%`。未转义 `%` 会注释掉行尾 `\\`，常见报错为：

```text
Misplaced \noalign
Missing \cr inserted
```

优先检查报错行上一行是否有未转义百分号。
