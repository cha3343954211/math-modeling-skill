# CUMCM 2015A 太阳影子定位案例记录

## 适用场景

当题目涉及太阳影子、直杆定位、由影尖轨迹反推地点/日期、或视频影子定位时参考本记录。典型问题是 2015A《太阳影子定位》。

## 核心模型

### 太阳位置模型

用太阳赤纬、时差方程和经度修正把北京时间转为真太阳时：

```text
δ = 23.45° sin(360°(284+n)/365)
B = 360°(n-81)/364
E_t = 9.87 sin(2B) - 7.53 cos(B) - 1.5 sin(B)
T_s = T + E_t + 4(λ - 120°)
ω = T_s/4 - 180°
sin α = sin φ sin δ + cos φ cos δ cos ω
```

太阳单位方向向量采用局部 East-North-Up 坐标：

```text
s_E = - cos δ sin ω
s_N = cos φ sin δ - sin φ cos δ cos ω
s_U = sin α
```

> 关键坑：东向分量符号要和时角正方向匹配。下午时角为正时，太阳在西侧，若 ENU 的 East 为正东，则 `s_E = -cosδ sinω`。符号写反会导致反演出南半球或明显不合理的地点，但残差仍可能很小，因为旋转角会吸收一部分错误。

影尖地理东-北坐标：

```text
r(t) = -H * (s_E, s_N) / s_U
L(t) = H / tan α(t)
```

### 附件坐标旋转

题面只定义 `xy` 平面，没有定义 x 轴朝向，因此反演时必须引入平面旋转角：

```text
p_i(φ, λ, H, θ) = R(θ) r(t_i; φ, λ, H, d)
min Σ ||p_i - q_i||²
```

优化变量：纬度 `φ`、经度 `λ`、`log(H)`、旋转角 `θ`。杆高用 `log(H)` 保证正值。

## 求解路线

### 问题1：已知地点、日期、杆高

正向逐分钟计算影长并保存曲线。2015-10-22 天安门 3m 直杆的复现结果：

- 09:00 影长约 `6.8841 m`
- 15:00 影长约 `7.0365 m`
- 最短影长约 `3.8411 m`，出现在 `11:58`

### 问题2：已知日期反演地点

1. 读附件1时间和影尖坐标。
2. 日期固定为 2015-04-18。
3. 多初值最小二乘拟合 `φ, λ, logH, θ`。
4. 输出经纬度、拟合杆高、坐标旋转角、RMSE、最大残差。
5. 用 bootstrap 或坐标扰动做稳定性分析。

一次可复现的候选结果：北纬 `18.3361°`、东经 `109.7018°`，拟合杆高 `1.9640 m`，RMSE `0.000251 m`。

### 问题3：未知日期反演地点与日期

1. 外层对全年日期做网格搜索，内层对地点/杆高/旋转角做最小二乘。
2. 先粗扫日期，再围绕低残差谷底细扫，避免 365 天 × 大量初值导致过慢。
3. 输出相隔足够天数的非重复候选，而不是只给一个唯一解。
4. 解释多解性：太阳赤纬在一年中近似对称，短时间影尖轨迹可能无法区分春/夏或冬/年初的等价日期。

一次可复现的候选结果：

- 附件2：2015-05-23，北纬 `39.7296°`、东经 `78.8739°`，RMSE `3.4e-05 m`；近似等价候选 2015-07-20，北纬 `39.8040°`、东经 `81.2112°`。
- 附件3：2015-11-16，北纬 `30.4121°`、东经 `106.5854°`，RMSE `0.000204 m`；近似等价候选 2015-01-24，北纬 `30.2418°`、东经 `113.2011°`。

## 问题4：视频缺失或可用时的处理

若视频可用：

1. 抽帧并记录帧时间。
2. 识别杆底和影尖；必要时少量人工校正关键帧。
3. 由已知杆高 2m 做像素--米标定；若透视明显，先做地面单应性校正。
4. 得到相对杆底的影尖坐标序列。
5. 若日期已知，用问题2模型；若日期未知，用问题3模型。
6. 用理论轨迹重投影到视频帧，检查像素残差和帧间平滑性。

若视频缺失：

- 不输出“可能拍摄地点”的伪结果。
- 在预检、论文摘要/问题四小节和最终交付说明中明确：当前目录没有视频本体，自动/非交互下载不可得。
- 给出完整模型流程，并说明补充视频后可直接运行同一反演框架。
