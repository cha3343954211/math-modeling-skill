# CUMCM 2012C 脑卒中发病环境因素分析案例记录

## 适用场景

当国赛/课程题涉及“疾病发病记录 + 气象/环境时序数据 + 干预建议”时参考。本记录来自 2012C 脑卒中发病环境因素分析的一次完整交付经验，重点是数据对齐、统计建模、风险解释、论文与支撑材料交付。

## 问题类型与推荐路线

2012C 属于典型的医学环境流行病学数据分析题，核心不是复杂黑箱预测，而是：

1. 从病例数据中提取发病时间、性别、年龄、职业等人口学特征；
2. 将发病日期与同期气象/污染等环境变量按日期对齐；
3. 识别发病人数的时间分布、季节性与高风险人群；
4. 建立环境因素与日发病数/风险水平之间的统计关系；
5. 输出可解释的干预建议。

推荐主线：

```text
数据审计与日期标准化
→ 病例人口学分层统计
→ 日期聚合（日发病数/周发病数/月发病数）
→ 气象环境数据按日期合并
→ baseline（均值/季节分层/简单泊松）
→ 相关分析 + GLM/泊松或负二项回归 + 滞后效应
→ 聚类/综合风险评分（可选）
→ bootstrap/时间切分验证 + 敏感性分析
→ 干预建议
```

## 数据处理要点

### 日期列标准化

病例和环境表中的日期可能混合出现 Excel 序列号、字符串、带时区时间戳。进入筛选、merge 或 groupby 前必须统一成无时区日期：

```python
def parse_date(v):
    import pandas as pd
    if pd.isna(v):
        return pd.NaT
    if isinstance(v, (int, float)):
        try:
            return pd.to_datetime(v, unit='D', origin='1899-12-30').normalize()
        except Exception:
            pass
    return pd.to_datetime(str(v), errors='coerce').tz_localize(None).normalize()
```

若 `pd.to_datetime(...).dt` 报时区或混合类型问题，优先检查是否存在 timezone-aware 与 naive datetime 混合；统一 `tz_localize(None)` 后再比较日期范围。

### 时间聚合口径

- 日发病数：用于 GLM、滞后效应、短期风险识别；
- 月/季节发病数：用于论文解释季节性与高发月份；
- 年龄段/性别/职业交叉表：用于高风险人群画像；
- 若病例记录包含多个诊断/地区字段，必须说明纳入与剔除规则。

## 建模建议

### Baseline

至少建立一个简单 baseline：

- 全期日均发病数；
- 按月份或季节的分层均值；
- 简单泊松模型（仅截距或仅季节项）。

主模型必须说明相对 baseline 的收益，例如 AIC/BIC 降低、MAE/RMSE 改善、残差更稳定或解释变量显著。

### 主模型候选

1. **相关分析**：Pearson/Spearman + 滞后 0--7 天窗口，用于初筛变量；
2. **GLM 泊松回归**：日发病数为计数响应变量，解释变量为温度、湿度、气压、风速、污染指标及滞后项；
3. **负二项回归**：若日发病数过度离散，优于泊松；
4. **Logistic/风险分级模型**：将高发日定义为超过分位数阈值，预测高风险日；
5. **PCA/熵权风险评分**：用于综合多个环境变量生成风险指数，但需和真实发病数验证。

### 滞后效应

医学环境题常存在延迟影响。建议对关键环境变量构造 `lag0`--`lag7` 或移动平均项：

```python
for col in env_cols:
    for lag in range(0, 8):
        df[f'{col}_lag{lag}'] = df[col].shift(lag)
```

论文中不要简单堆砌所有滞后项；应报告筛选标准（相关性、AIC、逐步选择或交叉验证）和最终保留变量。

## 验证与稳健性

- 计数模型：报告 MAE/RMSE、AIC/BIC、残差图、过度离散检查；
- 时间切分：用前期训练、后期验证，避免随机打乱造成时间泄漏；
- bootstrap：对日期或周进行重采样，检验变量贡献/风险排序稳定性；
- 敏感性：改变高发日阈值（如 75%、80%、90% 分位）或滞后窗口（3/5/7 天）观察结论是否稳定；
- 干预建议必须由模型结果支撑：例如针对高风险月份、年龄段、特定天气组合给出预警与健康管理策略。

## 论文写作重点

- 摘要必须给出：样本规模、最高风险人群/月份、显著环境因素、模型验证指标、干预建议；
- 数据预处理章节要写清日期对齐、缺失/异常、病例筛选口径；
- 结果章节至少包含：人口学分布表、月/季节发病趋势图、环境变量相关/回归结果表、风险等级或高发日识别图；
- “干预”不能泛泛写健康宣教，应对应模型发现，例如低温/高湿/气压突变前后的预警窗口、重点人群随访、社区筛查频率。
