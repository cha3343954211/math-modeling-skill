# 校赛 C 题无人机低空航线与空域调度案例记录

## 适用触发

当题目涉及无人机低空经济、低空空域网格化、航线覆盖、多无人机协同调度、冲突规避或作业效能评价时，可参考本案例路线。

## 题型拆解

典型四问结构可拆成：

1. **空域网格化评价**：区域重要度、作业密集度、冲突风险度 → 组合赋权 → 网格等级、容量、利用率。
2. **单机航线优化**：农田/矩形区域用覆盖路径规划；河道/电力线路用带状巡检路径；必须回扣续航、返航余量和机场接入距离。
3. **多机协同调度**：任务固定值守 + 优先级 + 同网格容量 + 水平/垂直安全间距 + 高度分层 + 时隙错峰。
4. **效能综合评价**：效率、安全、能耗、覆盖质量等指标，用 TOPSIS 或组合评价比较传统人工、随机无人机、优化调度三种模式。

## 可复用模型路线

### Q1 空域网格评价

- 网格边长可取 0.5 km 作为规划尺度下的平衡值。
- 指标：
  - 区域重要度：按任务优先级归一化；
  - 作业密集度：日均作业频次 / 作业面积；
  - 冲突风险度：任务重叠、城区随机性、管制带、邻近其他片区机场、禁飞区边界等。
- baseline：等权评分。
- 主模型：主观权重 + 熵权法组合赋权。
- 输出：每个网格的主作业类型、综合得分、容量等级、最大承载无人机数、利用率和适配性。

### Q2 单机航线

- 农田植保：Boustrophedon/往复条带覆盖，沿长边布设以减少转弯次数。
- 常规往返直线航线作为 baseline。
- 能耗模型可用：平飞能耗 + 转向/悬停附加能耗。
- 续航边界必须写清：

```text
最大航程 = 巡航速度 × 续航时间
有效航程 = 最大航程 × (1 - 安全余量)
单次最大有效覆盖面积 = 作业幅宽 × max(0, 有效航程 - 2 × 机场接入距离)
```

- 河道/线路巡检：按带状区域长度、有效巡检幅宽和端点换线距离估算无盲区航线，若单机不可完成，应明确分段接力而不是强行给出单机全线方案。

### Q3 多机调度

- baseline：随机同步起飞或无高度层分离方案，统计潜在冲突对。
- 主模型约束：
  - 同一空域单元同一时刻最多 N 架；
  - 水平安全间距；
  - 垂直安全间距；
  - 续航上限；
  - 任务优先级；
  - 所属片区机场起降。
- 常用可解释策略：
  - 高优先级任务先起飞；
  - 同一场景按容量分组，例如 3 架一组；
  - 不同场景采用高度层分离，层间距大于题目给定垂直安全间距；
  - 低优先级任务延后若干分钟错峰。
- 论文中要输出甘特图、冲突规避前后对比、覆盖率、完工时间、能耗均衡指标。

### Q4 综合效能评价

- 指标建议：作业效率、飞行安全、能耗成本、覆盖质量。
- 方法：TOPSIS；权重可结合题意设置，并做 ±20% 权重敏感性分析。
- 对比对象：传统人工、无人机随机航线、优化模型智能调度。

## 论文写作注意点

- 摘要必须按四问写出方法和关键数字。
- 对单机航线问题，若全域规模超过单机续航，应诚实写“单机单次任务包上限 + 多机/多架次接力”，不要伪造单架全域完成。
- 多机调度结果必须回代检查：高度层是否在 10–120m 内、层间距是否满足安全间距、同网格并行是否超容量、作业时长是否低于续航。
- 支撑材料中至少保存：网格表、航线表、机场辐射阈值表、调度表、效能评价表、frozen_numbers.json。

## 可视化建议

- Q1：网格散点/热力图 + 禁飞区框线；片区利用率条形图。
- Q2：优化航线与 baseline 的长度/能耗对比图；机场接入距离—有效覆盖面积曲线。
- Q3：32 架无人机调度甘特图；调度前后冲突对比柱状图。
- Q4：TOPSIS 贴近度条形图；权重敏感性折线图。

## 踩坑

- 论文图表和摘要数字只从 `results/frozen_numbers.json` 与最终表格读取；模型代码变动后必须重跑并重新编译 PDF。
- 打包邮件发送前要打开 zip 核验关键文件，不只看 zip 存在和大小。
